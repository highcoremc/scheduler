# Scheduler
Simple scheduler abstraction for paper/bungee plugins.


## How to use it in my project?

```xml
	<repositories>
		<repository>
		    <id>jitpack.io</id>
		    <url>https://jitpack.io</url>
		</repository>
	</repositories>
```

```xml
	<dependencies>
		<dependency>
		    <groupId>com.github.loperd</groupId>
		    <artifactId>scheduler</artifactId>
		    <version>Tag</version>
		</dependency>
	</dependencies>
```
